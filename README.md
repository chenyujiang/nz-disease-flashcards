# NZ Disease Flashcards / 新西兰常见病记忆卡

> 专为在新西兰学习、工作的医学生和医疗从业者设计的中英双语疾病记忆卡片应用。
> A bilingual (Chinese–English) disease flashcard app designed for medical students and healthcare professionals in New Zealand.

## 功能特色 / Features

- **60+ 种常见病** — 覆盖新西兰临床最常见疾病  
  **60+ common diseases** — covering the most prevalent conditions in NZ clinical practice
- **中英双语** — 每张卡片同时显示英文病名与中文译名  
  **Bilingual** — each card shows both the English name and Chinese translation
- **IPA 音标** — 英文病名附带国际音标，帮助正确发音  
  **IPA phonetics** — International Phonetic Alphabet pronunciation guide for every disease name
- **谐音记忆法** — 每种疾病配有中文谐音助记句，便于快速记忆  
  **Chinese mnemonic** — a phonetic memory hook in Chinese for rapid recall
- **卡片翻转** — 点击正面查看详情（主要症状 + NZ 临床备注）  
  **Flip cards** — click the front to reveal symptoms and NZ-specific clinical notes
- **多维筛选** — 按年龄（儿童 / 成人 / 老年）和系统分类筛选  
  **Multi-filter** — filter by age group (child / adult / elderly) and body system
- **关键词搜索** — 支持中英文搜索病名或助记内容  
  **Search** — supports Chinese and English keyword search
- **无需安装** — 纯静态 HTML 单文件，直接浏览器打开即用  
  **Zero setup** — single static HTML file, just open in any browser

## 使用方法 / How to Use

**中文：**
1. 下载 `index.html`
2. 用任意浏览器打开（无需服务器，无需网络连接）
3. 使用顶部工具栏按年龄或系统分类筛选卡片
4. 在搜索框输入中英文关键词搜索
5. 点击任意卡片翻转，查看主要症状和 NZ 临床备注

**English:**
1. Download `index.html`
2. Open in any browser — no server or internet connection required
3. Use the top toolbar to filter by age group or body system
4. Type Chinese or English keywords into the search box
5. Click any card to flip it and reveal symptoms and NZ clinical notes

## NZ 本地化内容 / NZ-Specific Content

每张卡片的临床备注均针对新西兰医疗体系整理。  
Clinical notes on every card are tailored to the New Zealand healthcare system, including:

- **PHARMAC 资助药物 / PHARMAC-funded medications** — NZ 公费处方药信息 / publicly funded drug information
- **免费筛查项目 / Free screening programmes** — BreastScreen Aotearoa、BowelScreen、宫颈筛查 / cervical screening 等
- **高发人群 / High-risk populations** — 毛利族（Māori）和太平洋岛民（Pacific peoples）的特殊风险数据 / specific risk data for Māori and Pacific peoples
- **NZ 发病率数据 / NZ epidemiology** — 基于新西兰本地流行病学数据 / based on local NZ statistics
- **公共医疗资源 / Public health resources** — ACC、公立医院、社区服务 / ACC, public hospitals, community services
